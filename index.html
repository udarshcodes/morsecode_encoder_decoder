Project Explanation

This project bridges the gap between historical communication and modern digital logic. It serves as a Bi-Directional Transceiver.

How it works:

  • Encoding: Type English text on this website. The Arduino converts it into Morse Code audio pulses (Buzzer) and Light signals.
  • Decoding: Use the tactile push-buttons on the hardware to tap dots and dashes. The system uses a time-based algorithm to decode the signals back into English text, displayed here in real-time.

Components List

  • Arduino Uno R3 (Microcontroller)
  • LCD 16x2 Display (Parallel Interface)
  • Piezo Buzzer (Audio Output)
  • Push Buttons x2 (Input: Dot & Dash)
  • Jumper Wires & Breadboard
  • USB Cable (Serial Communication)
